módulo "5-text_indentation"
==========================

O módulo em questão fornece a função "def text_indentation(text):"
---------------

Importamos a função:
>>> text_indentation = __import__('5-text_indentation').text_indentation

Caso 0, recebe um texto valido
>>> text_indentation("Lorem ipsum dolor sit amet")
Lorem ipsum dolor sit amet

Caso 1, receba um valor que não é uma string
>>> text_indentation(['textotexto'])
Traceback (most recent call last):
    ...
TypeError: text must be a string

Caso 2, texto valido e contem ., ? ou :
>>> text_indentation("Lorem ipsum dolor sit amet. alira")
Lorem ipsum dolor sit amet.
<BLANKLINE>
alira

Caso 3, Ignora espaço após quebra de linha e no inicio e fim do codigo
>>> text_indentation("  um. dois: três?    ")
um.

dois:
<BLANKLINE>
três?
<BLANKLINE>
